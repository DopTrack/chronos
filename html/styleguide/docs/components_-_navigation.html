found in: /source/css/project/components//_nav-tab.scss

Tabs are elements that help you organize and navigate multiple documents in a single container. They can be used for switching between items in the container. When target is not specified it will use the order of the DOM. It targets its direct siblings. You can break out your structure if you use a target attribute
