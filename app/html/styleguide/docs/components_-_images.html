images

found in: /source/css/project/base//_layout.scss

The photo component displays one ore more images in an content/detail page. The caption is always optional and will not show when the caption field is left blank. Bigger images will shrink down, and smaller images will shrink up. Make sure that if you're using a caption the image is no larger than 830px, for a proper side-by-side view

Image fullwidth without caption

found in: /source/css/project/base//_layout.scss

When the caption field is left blank, the image spans over 12 columns. Visually breaking the content paragraphs. The image does not automatically stretch to fill the space, so make sure the minimum width is around 1050
